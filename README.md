# Desafio Tunts.Rocks

## How to run?
1 - Clone the repository
```
git clone https://github.com/gph/desafio-tunts-rocks.git
```

2 - Install dependencies
```
npm install
```

3 - Run the application
```
npm start
```

4 - The spreadsheet will be created inside the app directory.
```
countries-list.xlsx
```
